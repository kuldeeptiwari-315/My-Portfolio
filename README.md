# Kuldip Tiwari · Portfolio

Modern single-page portfolio showcasing Kuldip Tiwari’s skills, featured projects, and contact info.

## Tech Stack
- HTML5 layout with semantic sections
- Custom CSS (glassmorphism + neon gradients)
- FormSubmit contact form integration

## Local Development
1. Clone/download this repo.
2. Open `Index.html` directly in a browser or start a lightweight server:
   ```bash
   npx serve .
   ```
3. Edit `style.css` / `Index.html`, then refresh to preview.

## SEO & Performance
- Includes meta description, Open Graph/Twitter tags, canonical URL, JSON-LD schema.
- Fonts preconnected/preloaded, JS deferred, hero image optimized for CLS, nav highlighting via IntersectionObserver.
- `robots.txt` + `sitemap.xml` provided; update the URLs to your deployed domain.

## Deployment Checklist
- Replace placeholder domain (`https://kuldeeptiwari.dev/`) with the live URL in meta tags, JSON-LD, and `robots.txt`.
- Compress `kt.jpg` for faster load.
- Generate and upload `sitemap.xml`, then submit both sitemap and site to Google Search Console/Bing Webmaster Tools.
